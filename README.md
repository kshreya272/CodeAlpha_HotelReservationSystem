# CodeAlpha - Hotel Reservation System

This project is developed as part of the CodeAlpha Java Programming Internship.

## Project Description
The Hotel Reservation System is a Java-based console application that allows users to view available rooms, book hotel rooms, cancel reservations, and manage booking details.

## Features
- View available hotel rooms
- Book a room
- Cancel room booking
- Display room details
- Payment simulation
- Menu-driven console application

## Technologies Used
- Java
- VS Code
- Object-Oriented Programming (OOP)
